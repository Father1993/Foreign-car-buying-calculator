Друзья привет, калькулятор для расчета стоимости авто под ключ <br>
Калькулятор может посчитать как автомобили из Японии(правый руль), так и автомобли с Южной Кореи <br>
А так-же поможет вам высчитать вашу ставку в йенах, исходя из вашего бюджета (для покупки авто с аукционов Японии)<br>
<br>
Запускаеться локально из любого файла с расширением .html, навигация осущесвляеться по ссылкам с любого файла.<br>
<br>

* 
переменные: 
expensesJp = расходы по Японии в йенах, можно отредактировать в коде файл- main.js(для расчета стоимости в рублях)
ex в файле- rev.js(для расчета ставки в йенах)
